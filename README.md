# nuNet

nuNet is a convolutional neural network platform for reconstructing neutrino properties from radio emission produced in Askaryan events

## Folder structure

    .
    ├── ...
    ├── training                # directory for training code
    │   └── runs                
    │       ├── direction       
    │       │   ├── archive     
    │       │   ├── base
    │       │   └── runD[n]     # n = 1, 2, 3, ...
    │       ├── energy
    │       │   ├── archive
    │       │   ├── base
    │       │   └── runE[n]     # n = 1, 2, 3, ...
    │       └── flavour
    │           ├── archive
    │           ├── base
    │           └── runF[n]     # n = 1, 2, 3, ...
    └── ...
